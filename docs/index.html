The easiest way to mock/stub API endpoints for rapid prototyping and development.

Get started

Easy to Use

Mocktail is a zero-code solution meaning that in just a few clicks you are able to start developing against real endpoints.

Versatile

You can design your response bodies to be as complex or as simple as you like. We allow you to use a mixture of static and dynamic values from a whole host of data types.

Minimum Setup

Under the hood we run a lightweight web server that needs almost no configuration. We do all the heavy lifting so you can get on with your development.
