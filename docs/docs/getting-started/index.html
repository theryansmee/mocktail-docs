What is Mocktail?​

Mocktail is a simple desktop application for Mac & Windows that allows you to quickly mock/stub endpoints.

Adding domains to the domain whitelist​

Once you have Mocktail downloaded, all you need to do before you get started is add your request urls the domain whitelist. If you are making requesting to Mocktail from localhost on ports 4200, 8080 or 3000 then you can skip this step as there are added to the whitelist by default.

Click the "App Settings" button at the bottom fo the left hand side menu

Click the "Add domain" button under the domain white list section, and add the domain.

Click the "Save" button in the bottom right hand side of the screen. If successful you will be prompted to restart Mocktail.
